## 1. DRILL DE IDENTIFICACIÓN: ¿ACTIVO O PASIVO?

**Instrucción**: Clasifica los siguientes ítems usando la regla: *"¿Me pone dinero o me saca dinero?"*.

| Ítem | Tu Respuesta (Activo/Pasivo) | ¿Por qué? |
| :--- | :--- | :--- |
| Tu coche personal (pagado) | | |
| Una casa que alquilas a otros (rentable) | | |
| Tu colección de Funko Pops | | |
| Acciones de Apple ($100 usd) | | |
| Tarjeta de Crédito (Saldo $0) | | |
| Tarjeta de Crédito (Saldo -$500) | | |

<details>
<summary>Ver Solución</summary>

* **Coche**: PASIVO. Gasta seguro y gasolina.
* **Casa en alquiler**: ACTIVO. Genera renta.
* **Funkos**: PASIVO (o activo especulativo de muy bajo valor).
* **Acciones**: ACTIVO. Generan dividendos o valor.
* **TC en cero**: NEUTRO (Herramienta).
* **TC con deuda**: PASIVO (Intereses).

</details>

## 2. CASO DE ESTUDIO: LAS FINANZAS DE ANA

Ana cree que le va bien. Gana $2,000 al mes.
Ayúdala a calcular su **Patrimonio Neto real**.

**Su Lista:**

* Dinero en cuenta sueldo: $500
* Coche (valor de venta): $8,000
* Préstamo del coche (lo que debe): $10,000
* Ropa y Zapatos (precio de compra): $3,000 (Precio venta real: $300)
* Deuda de Tarjeta de Crédito: $2,000
* Fondo de Inversión: $1,000

**Cálculo:**

1. Total Activos Reales: $______
2. Total Pasivos: $______
3. Patrimonio Neto: $______
